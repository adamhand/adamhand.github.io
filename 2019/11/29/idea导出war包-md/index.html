idea导出war包.md

发表于 2019-11-29 | 分类于 工具

有两种方法，第一种，如下图所示，首先选中Project Structure -> Artifacts -> Web Application Exploded -> From Modules..，点击下面的确定按钮。



接着，选中Project Structure -> Artifacts -> Web Application Archive -> From ...，点击下面的确定按钮。



接着点击任务栏的Build -> Build Artifacts即可编译得到war包。

Exploded和Archive的区别是，前者生成文件夹，后者生成war包。如果不先生成Web Application Exploded，Web Application Archive下面只有一个Empty，这时如果直接点确定按钮，然后编译的话，是编译不出war包的。这就有了方法二。

不先生成Exploded，而是直接生成Web Application Archive的Empty的话， 如下图所示。



这时，需要手动添加jar包。如下图所示。点击右侧的Available Elements，选中需要添加的文件夹，右键选择Put into Output Root，之后再Build即可得到war包。
