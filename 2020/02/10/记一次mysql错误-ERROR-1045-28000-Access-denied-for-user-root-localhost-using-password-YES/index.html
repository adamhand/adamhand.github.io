打开另一个cmd窗口，输入mysql -uroot -p，随便输入密码即可登录。登录后重新修改一下mysql的密码，先将其置为空：

关闭第一个cmd窗口，启动mysql服务，之后输入mysql -u root -p，由于密码为空，直接回车即可登录。之后使用ALTER USER 'root'@'localhost' IDENTIFIED BY '新密码';更改密码，之后退出即可，下次即可正常登录。
